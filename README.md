# Task Management App

A simple command-line Task Management Application built using Python.  
This project allows users to add, update, delete, and view daily tasks efficiently.

## Features
- Add new tasks
- Update existing tasks
- Delete tasks
- View all tasks
- User-friendly menu system

## Technologies Used
- Python

## Concepts Used
- Functions
- Lists
- Loops
- Conditional Statements

## How to Run
1. Install Python
2. Download the project
3. Run the file:

```bash
python Task_Management_app.py
